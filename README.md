## Usage

1. Set your preferred theme using [catppuccin/slack](https://github.com/catppuccin/slack).
1. Install Stylus [Firefox](https://addons.mozilla.org/firefox/addon/styl-us/)/[Chrome](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) extension.
1. [Click here to install.](https://github.com/jcyamo/catppuccin-slack/raw/main/catppuccin.user.css)
